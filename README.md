## Featured Projects

<details>
<summary><b>🤟 Sign SaaS — Live Sign Language Translator</b></summary>
<br/>

A web platform that translates sign language in real time and helps users learn it, built solo using Google's Antigravity agentic development platform with Firebase authentication.

| Category | Details |
|---|---|
| **Stack** | TypeScript, HTML, CSS, Python (FastAPI backend) |
| **Type** | Solo project — SaaS-style with a free and premium tier |
| **Key Features** | Real-time sign language translation, a learning mode, and a premium tier for uploading and translating pre-recorded videos |
| **Auth** | Firebase Authentication |
| **Repository** | [github.com/Mohammad-Amaan-A/Sign-SaaS](https://github.com/Mohammad-Amaan-A/Sign-SaaS) |

Built using an AI-assisted, agent-first development workflow, then reviewed and refined manually. Currently working back through the translation pipeline to build a deeper first-hand understanding of it — an active priority before this project comes up in an interview.

</details>
